# NZBooks V2 Frontend-Only

This is the frontend-only test build of NZBooks V2.

Use it to test:
- UI
- voucher flow
- posting logic
- ledger
- trial balance
- profit & loss
- balance sheet
- PDF / Excel export buttons

## No backend required
This version does not use Supabase and does not require environment variables.

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
Upload directly to GitHub and deploy. No backend setup needed.

## Important
This version uses in-memory demo data only. Refreshing the app resets the data.
