# NZBooks Posting Rules

1. A voucher must contain at least two lines.
2. Total debit must equal total credit.
3. Draft vouchers can be edited.
4. Posted vouchers cannot be edited.
5. Posted vouchers cannot be deleted.
6. Reversal creates a new voucher with opposite values.
7. Trial Balance uses posted vouchers only.
8. General Ledger uses posted vouchers only.
9. Profit & Loss uses Income and Expense accounts.
10. Balance Sheet uses Asset, Liability, and Equity accounts.
11. If a fiscal year lock date exists, vouchers on or before that date cannot be added or posted.
