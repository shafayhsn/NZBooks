# NZBooks V6 Reports Core

Focused frontend-only build for the 3 critical legacy report families:

1. Sub Ledger Balance
2. Voucher Prints
3. Account-wise Ledger

## Goal
This build is designed for report testing first.
It prioritizes print/export layout over general app UI.

## Run
```bash
npm install
npm run dev
```

## Includes
- old-style printable report layouts
- PDF export
- Excel export
- report selector
- voucher type selector
- account selector

## Important
This is still a testing build with demo data.
It is designed to let you validate layout direction and report behavior quickly.
